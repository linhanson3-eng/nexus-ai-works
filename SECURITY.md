# 安全策略

## 报告漏洞

如果你发现安全漏洞，请**不要**提交公开 Issue。

请发送邮件至项目维护者，描述漏洞详情和复现步骤。我们会在 48 小时内确认收到报告，并在 7 个工作日内给出评估和修复计划。

## 支持版本

| 版本 | 支持状态 |
|------|---------|
| v1.0.x | 活跃支持 |

## 安全模型

Nexus AI Works 采用多层安全防护：

- **Agent 执行层**: Shell 命令白名单 + 禁止模式检测
- **路径访问层**: 工作区沙箱 + 路径穿越防护
- **工具权限层**: 按 Agent 角色过滤可用工具
- **密钥管理层**: 硬编码密钥检测 (9 种模式)
- **预算控制层**: Token / 成本 / 工具调用 / 会话轮次上限

更多细节见 [ARCHITECTURE.md](ARCHITECTURE.md) 安全模型章节。

## 上报处理流程

1. 收到报告 → 48h 内确认
2. 7 个工作日内评估并修复
3. 修复后发布安全公告
4. 在 CHANGELOG 中公开致谢（征得报告者同意后）
