# 隐私政策

最后更新: 2026-05-23

## 1. 信息收集

### 我们收集的信息
- **账户信息**: 用户名、密码哈希 (PBKDF2-HMAC-SHA256)
- **使用数据**: 工作区数量、工作流执行次数、agent 调用次数
- **配置数据**: LLM API Key、搜索 API Key（可选）

### 我们不收集的信息
- LLM API 调用内容（由第三方 LLM 提供商处理）
- 工作区内的具体文件内容
- Agent 对话历史（除非您主动开启记忆功能）

## 2. 信息存储

- 所有数据存储在本地 `~/.nexus/` 目录
- 密码使用 PBKDF2 100,000 次迭代加密
- API Key 以文件权限保护存储

## 3. 信息使用

- 提供和改善服务
- 计费和订阅管理
- 安全审计和滥用防护

## 4. 信息分享

我们不向第三方出售用户数据。仅在以下情况分享：
- 法律法规要求
- 保护平台安全所必需
- 经您明确同意

## 5. 数据安全

- 密码使用行业标准加密存储
- API Key 以 chmod 600 权限保护
- 建议用户不要分享 API Key

## 6. 您的权利

- 访问您的个人数据
- 更正不准确的数据
- 删除您的账户和相关数据
- 导出您的数据

## 7. 联系我们

隐私相关问题请通过 Issue 或邮件联系项目维护者。
